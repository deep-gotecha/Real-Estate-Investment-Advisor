#  Real Estate Investment Advisor

**Predicting Property Profitability & Future Value**

---

# 1.  Project Overview

This project is an **end-to-end Machine Learning application** designed to assist real estate investors in making **data-driven decisions**.

The system performs two key tasks:

* **Classification** → Determines whether a property is a *Good Investment* or not
* **Regression** → Predicts the *future price of the property after 5 years*

The application is deployed using **Streamlit**, allowing users to input property details and instantly receive predictions along with **data insights (EDA dashboards)**.

---

# 2.  Problem Statement

Real estate investment decisions are often:

* Subjective
* Based on incomplete information
* Influenced by market speculation

This creates challenges such as:

* Difficulty in identifying profitable properties
* Lack of future price estimation
* No structured decision-making framework

###  Objective

To build a system that:

1. Classifies properties into:

   * Good Investment
   * Not Good Investment

2. Predicts:

   * Future price (5-year horizon)

3. Provides:

   * Data-driven insights using EDA
   * A user-friendly interface for decision-making

---

# 4.  Project Workflow (End-to-End)

---

##  Step 1: Data Loading & Cleaning

* Loaded dataset using **Pandas**
* Removed duplicates
* Handled missing values:

  * Numeric → filled with 0 or median
  * Categorical → replaced with `"Unknown"`

---

##  Step 2: Feature Engineering

Created meaningful features to improve model performance:

###  Age of Property

```text
Age = Current Year - Year Built
```

###  Price per SqFt

```text
Price per SqFt = Price / Size
```

###  Investment Label (Classification Target)

Defined using business logic:

```text
If Price ≤ Median Price → Good Investment (1)
Else → Not Good Investment (0)
```

###  Future Price (Regression Target)

Using compound growth:

```text
Future Price = Current Price × (1 + r)^t
Where:
r = 8% growth rate
t = 5 years
```

---

##  Step 3: Data Preprocessing

Used **ColumnTransformer + Pipeline**:

### Numerical Features:

* StandardScaler applied

### Categorical Features:

* OneHotEncoder used
* Handles unseen categories safely

---

##  Step 4: Model Building

Two models were trained:

###  Classification Model

* Algorithm: Random Forest Classifier
* Task: Predict Good Investment

###  Regression Model

* Algorithm: Random Forest Regressor
* Task: Predict Future Price

---

##  Step 5: Pipeline Creation

Built a **machine learning pipeline**:

```text
Input Data → Preprocessing → Model → Output
```

This ensures:

* Consistent transformations
* No manual preprocessing during prediction

---

##  Step 6: Model Saving

Saved trained models using **Joblib**:

* `classifier_pipeline.pkl`
* `regressor_pipeline.pkl`
* `feature_columns.pkl`

This allows:

* No retraining required
* Direct deployment

---
